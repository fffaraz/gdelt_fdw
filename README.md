# GDELT_FDW
Dockerized Multicorn PostgreSQL Foreign Data Wrapper (FDW) for Retrieving [GDELT](https://www.gdeltproject.org/) events database.

### How to Run
```
docker-compose up
chmod 777 volumes/data
```
